# ጥቁር ቡና (Tikur Buna) ☕

A modern bilingual Ethiopian coffee shop website that showcases the rich culture and taste of Ethiopian coffee.

## 🌟 Features

- Responsive design for mobile and desktop
- Amharic & English content
- Coffee menu and categories
- Ethiopian coffee ceremony section
- About, Location, and Contact sections
- Modern black, brown, and gold theme

## 🛠️ Built With

- React
- Vite
- TypeScript
- Tailwind CSS
- shadcn/ui
